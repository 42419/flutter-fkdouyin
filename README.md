# 抖音解析工具 (FKDouyin Flutter)

一款基于 Flutter 开发的现代化短视频解析工具，支持多平台视频无水印下载。致力于提供简洁、流畅、优雅的用户体验。

## ✨ 主要功能

- **无水印下载**：支持解析并下载无水印的高清视频。
- **多清晰度选择**：提供多种分辨率和帧率选项，满足不同需求。
- **历史记录**：自动保存解析历史，方便随时回看和重新下载。
- **暗黑模式**：完美适配系统深色模式，夜间使用更舒适。
- **自动更新**：内置应用内更新检测，通过代理加速下载最新版本。
- **现代化 UI**：采用 Material Design 3 设计语言，界面美观大方。

## 🚀 开始使用

### 安装

前往 [Releases](https://github.com/42419/flutter-fkdouyin/releases) 页面下载最新版本。

### 开发构建

1. **环境要求**
   - Flutter SDK >= 3.3.0
   - Dart SDK >= 3.3.0

2. **获取代码**

   ```bash
   git clone https://github.com/42419/flutter-fkdouyin.git
   cd flutter-fkdouyin
   ```

3. **安装依赖**

   ```bash
   flutter pub get
   ```

4. **运行项目**

   ```bash
   flutter run
   ```
